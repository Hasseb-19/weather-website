# Weather Website

A simple JavaScript weather website that shows the current weather for any city using the OpenWeatherMap API.  

---

## Features

- Search for any city and see the current weather
- Displays:
  - City name
  - Temperature (Â°C)
  - Weather condition
  - Humidity (%)
  - Wind speed (mph)
- Dynamic weather icons based on the condition
- Autocomplete suggestions removed for simplicity

---

## Technologies Used

- HTML
- CSS
- JavaScript (DOM & Fetch API)
- OpenWeatherMap API

---

## How to Use

1. Open `Weather.html` in a browser.
2. Type the name of a city in the search bar and press **Search**.
3. The weather card updates with the city's current weather.

---

## Notes

- The app uses a free OpenWeatherMap API key. You can replace it with your own key.
- Icons are stored locally in the `Icons` folder.
- Wind speed is displayed in **mph**.
